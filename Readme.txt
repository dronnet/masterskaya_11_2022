# Исследование 

двух датасетов для предсказания сердечных заболеваний.
 
Контекст: основной причиной смерти в развитых странах являются болезни сердца. Поэтому необходимо проделать работу, чтобы помочь предотвратить риск сердечного приступа или инсульта.

датасеты взяты отсюда:

https://www.kaggle.com/datasets/rishidamarla/heart-disease-prediction

https://www.kaggle.com/datasets/redwankarimsony/heart-disease-data

-- 28_11_2022 --
Два датасета, в итоге, были склеены вместе в один, чтобы получить больше данных для исследования и построения модели.
Проведена очистка данных от экстремальных значений. Проведена обработка пропущенных данных.
Проведена работа по замене значений в столбцах на одинаковые для двух датасетов.
Для исследования данных ппостроены различные диаграммы и проверены корреляции в данных.

После склеивания, данные в получившимся датасете были перемешаны и удалены дубликаты.

Датасет разделен на обучающую и тестовую выборку в соотношении 70% / 30%.
Проверены размеры получившихся выборок и сбалансированность классов.

Выбран целевой признак и метрика для оценки качества моделей.

Проведено обучения четырёх моделей DecisionTreeClassifier, RandomForestClassifier, LogisticRegression, SVM.

Наилучшие результаты показала модель "случайный лес" RandomForestClassifier, её рельтат f1_score, после настройки порога составил 0.854.

Стоит ещё проверить идею "нормирования параметров датасета". Возможно результат будет выше чем f1_score = 0.854
